# 林彥成 | 前端三分鐘

嗨，我是林彥成，10 年網站開發經驗的前端工程師。

現專注於 React.js 技術與應用，熟悉從前端到後端的開發流程，包含 Node.js 與 Socket.IO 的 API 開發與即時資料交換。

起初是後端工程師，後來因為對「使用者經驗」的興趣與追求，走進前端的世界，也因為喜歡教學與分享，把學習的心得記錄成部落格，成立了【前端三分鐘】這個個人技術筆記與分享平台。
